## Installation

### Doom Emacs

in `packages.el`

``` lisp
(package! hurl-mode :recipe (:host github :repo "Orange-OpenSource/hurl"))
```

### straight.el

``` lisp
(straight-use-package
 '(hurl-mode :type git :host github :repo "Orange-OpenSource/hurl"))
```
